CardPaySuite UI - Phase 1
------------------------
Files included:
- index.html (homepage with header, hero, cards, footer)
- about.html
- learn/index.html
- tools/index.html
- css/style.css

How to use:
- Upload the folder 'CardPaySuite_UI_Phase1' to your GoDaddy Web Hosting (public_html or root)
- Ensure 'css/style.css' is reachable at css/style.css relative to pages
- Replace placeholder text and add real content in learn/ and tools/ folders
